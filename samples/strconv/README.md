2. [文字列をbool型に変換する(ParseBool)](./strconv_sample_002.go)
3. [文字列を数値に変換する(ParseInt、ParseUint、ParseFloat)](./strconv_sample_003.go)
4. [数値を文字列に変換する(Itoa)](./strconv_sample_004.go)
5. [bool型を文字列に変換する(FormatBool)](./strconv_sample_005.go)
6. [数値を文字列に変換する(FormatInt、FormatUint、FormatFloat)](./strconv_sample_006.go)
